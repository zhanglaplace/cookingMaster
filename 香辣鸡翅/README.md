## 香辣鸡翅
==============

### 材料
----------------

  鸡翅：6-8个
  
  蒜蓉
  
  姜蓉
  
  干辣椒粉
  
  盐
  
  生抽
  
  生粉
  
  水
  
  
### 制作过程
----------------
1. 去血水：

    生鸡翅洗净，清水煮，起泡沫时捞出来滤干。
    
2. 腌制鸡翅:

    鸡翅中加入生抽2勺、盐少许，生粉2勺子抓均匀，腌制，时间看自己而定，一般10-30分钟即可。
  
    此时可以将蒜和姜切成蓉，待用
  
3. 初炒 :

    小锅中放底油，待油烧至七分熟时，鸡翅放入，煎至两边焦黄（不是烤焦），捞起
  
  
4. 继续炒:

    上一步中的油不用倒掉，加入蒜蓉和姜蓉入锅中爆香，加入2勺辣椒粉，加盐调味，倒入上一步的鸡翅，翻炒1-2分钟即可。
  
  
5. 可选步骤:

    倒入一点生抽，加2勺子水，盖上锅盖焖1分钟收汁 出锅。。
